ЭТАП 0 — Фундамент (сделаем сегодня-завтра)
Цель: чистое место для работы, где ничего не сломано.

Шаг 0.1 — GitHub и окружение
Создать репозиторий на GitHub (назовём losiny-map)

Клонировать к себе на компьютер

Установить Node.js (если нет) — версию LTS

Шаг 0.2 — Проект Vite + React
bash
npm create vite@latest losiny-map -- --template react
cd losiny-map
npm install
npm run dev
Увидеть надпись "Vite + React"

Шаг 0.3 — Структура папок (сразу правильная)
text
src/
  components/     # кнопки, панели, попапы
  hooks/          # свои React-хуки (GPS, офлайн)
  services/       # работа с картой, данными, хранилищем
  store/          # Zustand (состояние)
  data/           # GeoJSON и PMTiles файлы (положим рядом)
  utils/          # Turf.js обёртки
Создать эти папки (пока пустые)

Шаг 0.4 — Установка зависимостей (все сразу, чтобы не мучаться)
bash
npm install maplibre-gl @maplibre/maplibre-gl-geolocate
npm install pmtiles zustand
npm install @turf/turf
npm install idb          # удобная работа с IndexedDB
npm install vite-plugin-pwa   # для офлайн-режима
Команды выполнились без ошибок

Шаг 0.5 — PWA заглушка (чтобы потом не переделывать)
Добавить vite-plugin-pwa в vite.config.js

Создать manifest.json (иконки сделаем позже)

Результат этапа 0: ✅ проект запускается, структура готова, зависимости на месте.
